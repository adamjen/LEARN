# memory_bank/project-brief.md

**Project**: ARC & Tone Scale Learning  
**Type**: Educational Web Application  
**Stack**: TypeScript, React, Vite, Tailwind CSS, Vitest

**Core Concepts**:

- ARC Triangle (Appreciation, Reality, Communication)
- Tone Scale (-40 to +40 emotional gradient)
- Emotional Intelligence (Goleman, Six Seconds, Freedman-Fariselli models)

**Key Terminology**:

- Appreciation = Value/care/regard for others
- Reality = Shared understanding/truth
- Communication = Information exchange

**Content Guidelines**:

- Use "Appreciation" (not "Affinity")
- Australian English only
- Follow L. Ron Hubbard historical context objectively
- Maintain neutral, educational tone

**Tech Stack**:

- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS
- State: Zustand (implied from store files)
- Testing: Vitest
- UI Components: Custom React components
